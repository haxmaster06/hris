# UNIVERSAL TABLE STANDARD

Seluruh tabel wajib memiliki field berikut:

```sql
id UUID PRIMARY KEY

tenant_id UUID

created_at TIMESTAMP

updated_at TIMESTAMP

deleted_at TIMESTAMP NULL

created_by UUID

updated_by UUID

deleted_by UUID NULL

version INTEGER
```
